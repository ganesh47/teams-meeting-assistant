% teams-meeting-assistant(1)

# NAME

teams-meeting-assistant - Linux CLI for Teams join automation and local transcript generation

# SYNOPSIS

**teams-meeting-assistant** `linux-cli <teams-link>`

**teams-meeting-assistant** `join <teams-link>`

**teams-meeting-assistant** `offline-pipeline [mock|faster-whisper]`

# DESCRIPTION

A Linux-oriented CLI for experimenting with Microsoft Teams join automation and local transcript generation.

# OUTPUTS

Transcript artifacts are written under `artifacts/<session-id>/`.
